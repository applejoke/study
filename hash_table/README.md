# 哈希表项目

## 工作流程

### 在 Trae 中编写代码
1. 在 Trae 中编辑代码
2. 使用 Git 提交更改：
   ```bash
   git add .
   git commit -m "描述你的修改"
   git push
   ```

### 在 VS2022 中调试
1. 打开 VS2022
2. 克隆或拉取最新代码：
   ```bash
   git pull
   ```
3. 创建或打开项目
4. 按 F5 开始调试

## 项目结构
```
hash_table/
├── code/           # 源代码
│   ├── Hash.c
│   └── main.c
├── include/        # 头文件
│   └── Hash.h
├── .vscode/        # VSCode 配置（Trae 使用）

```

## Git 快速命令
```bash
# 查看状态
git status

# 提交更改
git add .
git commit -m "描述"
git push

# 拉取更新

```
